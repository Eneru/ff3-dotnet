## Run Locally

Clone the project

```bash
  git clone https://github.com/Eneru/ff3-dotnet.git
```

Go to the project directory

```bash
  cd ff3-dotnet
```

Build the library

```bash
  dotnet build
```


## Running Tests

To run tests, run the following command

```bash
  dotnet test
```
